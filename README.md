# AZ-104 Hands-On Lab

Practice labs for the Microsoft Azure Administrator (AZ-104) certification exam using real Azure resources.

## Structure

```
lab-tracker.md          # Progress tracker across all labs
labs/
  lab-01-*.md           # Individual lab files with scenario, tasks, and results
  lab-02-*.md
  ...
```

## Workflow

1. A new lab is assigned covering an AZ-104 domain
2. Complete the tasks in the Azure portal
3. Work is verified via Azure CLI (`az` commands)
4. Lab file is updated with pass/fail results

## AZ-104 Domains

- Identity & Governance
- Storage
- Compute
- Networking
- Monitoring & Backup

## Prerequisites

- Azure subscription
- Azure CLI installed and authenticated (`az login`)
